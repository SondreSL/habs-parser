habs-parser-0.0.1: The parser of the Haskell-ABS (habs) backend

A transcompiler from the ABS language to Haskell. This Haskell package includes both the ABS stdlib & ABS runtime expressed in Haskell, as well as the transcompiler to translate ABS code to Haskell code.

ABS developers are advised to check the online documentation of the BNF grammar for ABS
